# VLADDOS MAX (Immersive Portfolio)

## Запуск

```bash
npm i
npm run dev
```

Открой `http://localhost:3000`.

## Ассеты

Все картинки лежат в `public/assets/hero/`:
- `bg.png` — фон-сцена (твоя комната)
- `logo.png` — подпись/логотип VLADDOS
- `character.png` — оверлей персонажа для глубины

Аудио (опционально):
- `public/assets/audio/ambient.mp3`
- `public/assets/audio/click.mp3`

Если аудио нет — кнопка SOUND будет работать, но звука не будет.

## Что дальше

- Подставить реальные проекты/контент в `/projects`, `/resume`, `/contacts`.
- Можно добавить "сцены" (dots справа снизу) и переключение фона.
- Можно разрезать фон на слои и получить ещё более сильный параллакс.
